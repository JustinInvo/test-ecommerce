Hola! 

Para levantar el proyecto puedes seguir estos pasos:
1. npm i
2. Agregar las variables de entorno
2.1. Las variables de entorno no se deben de subir al repositorio,
    este caso será una excepción ya que no hay información sensible
    y es una prueba :)

    NEXT_PUBLIC_SITE_URL=https://fakestoreapi.com
    NEXT_PUBLIC_API_BASE_URL=https://fakestoreapi.com
    NEXT_PUBLIC_COPY_YEAR=2026

3. npm run dev
